# 💫 @brejcha13320/design-system-bootstrap

✨ **Sistema de diseño basado en Bootstrap listo para usar desde npm**.

Este paquete ofrece estilos y componentes personalizados construidos sobre el framework **Bootstrap** para un desarrollo UI consistente, accesible y escalable. Aprovecha toda la potencia de Bootstrap mientras mantienes un lenguaje visual propio.

> 💡 Compatible con proyectos modernos que usen Bootstrap como base para UI y componentes.

---

## 🚀 Instalación

Instala el paquete desde npm:

```bash
npm install @brejcha13320/design-system-bootstrap
```
